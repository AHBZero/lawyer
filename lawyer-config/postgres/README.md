# Postgres
======

## Configuração do postgres no JBoss AS 7.1.1

Inserir o datasource em  standalone-datasource.xml no standalone do JBoss

Inserir a pasta org/postgres/main na pasta modules do JBoss AS

### Não criei o xml para subir com a aplicação porque sua senha deve ser diferente da minha, dai a cada commit sem querer vai ficar alterando e tal.
